## 🚀 快速启动

### 方法1: 使用启动脚本（最简单）

```bash
# 在项目根目录
./start_all.sh
```

这将自动启动前后端服务。

### 方法2: 手动启动

**步骤1：启动后端**
```bash
# 在项目根目录
python main.py
```

**步骤2：启动前端**
```bash
# 新开一个终端
cd frontend
python -m http.server 8080
```

**步骤3：访问**
在浏览器打开: http://localhost:8080

## 🔧 配置说明

### API地址配置

编辑 `frontend/app.js` 第一行：

```javascript
const API_BASE_URL = 'http://localhost:8000';
```

如果后端运行在不同地址或端口，修改此配置。

### 端口配置

默认前端运行在 8080 端口，可以修改：

```bash
# 使用其他端口

```

## 💻 使用说明

### 1. 智能助手使用

**方式A：快捷按钮**
1. 点击欢迎页面的快捷按钮
2. 自动填充常用查询
3. 发送获取结果

1. 在输入框输入查询
2. 按 Enter 或点击发送

**示例查询：**

1. 切换到"酒店预订"标签
2. 输入目的地
3. 设置预算（可选）

5. 点击"搜索酒店"
6. 浏览酒店列表

### 4. 景点推荐使用

1. 切换到"景点推荐"标签
2. 输入目的地
3. 选择旅行天数
4. 勾选兴趣偏好
5. 点击"推荐景点"
6. 查看推荐结果

## 🔍 调试技巧

### 1. 查看控制台

打开浏览器开发者工具（F12），查看：
- Console 标签：JavaScript错误
- Network 标签：API请求
- Elements 标签：DOM结构

### 2. 常见问题

**问题：页面空白**
```
解决：查看控制台是否有JavaScript错误
```

**问题：API无响应**
```
解决：
1. 检查后端是否启动
2. 检查API地址配置
3. 查看Network标签的请求状态
```

**问题：样式不正常**
```
解决：
1. 清除浏览器缓存
2. 检查styles.css是否加载
3. 查看控制台CSS错误
```

### 3. 调试模式

在 `app.js` 中添加调试日志：

```javascript
console.log('API响应:', data);
```

### 修改主题色

编辑 `styles.css` 中的 CSS 变量：

```css

    --primary-color: #2563eb;     /* 主色 */
```


## 🔐 安全考虑

### 生产环境建议

1. **使用HTTPS**
   ```
   确保前后端都使用HTTPS协议
   ```

2. **限制CORS**
   ```python
   # 后端只允许特定域名
   allow_origins=["https://yourdomain.com"]
   ```

3. **添加认证**
   ```javascript
   // 前端添加token
   headers: {
       'Authorization': 'Bearer ' + token
   }
   ```

4. **输入验证**
   ```javascript
   // 前端验证用户输入
   if (!validateInput(input)) {
       showError('输入格式不正确');
       return;
   }
   ```
